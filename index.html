O que é HTML? (Voltar)

É uma sigla, liguagem de marcação de hipertexto.

H- Hyper

T- Text

M- Market

L- Language

- Linguagem: Forma de comunicação. No nosso caso, entre o programador e o navegador de Internet (Google Chrome, Firefox, Safari, Edge, Internet Explorer…)
- Marcação: O HTML ‘marca’ o que é o que em uma página. Ele que vai dizer o que é um texto, o que é uma imagem, um vídeo e assim por diante.
- Hypertexto: faz referência a escritá gráfica, sintetizando, é um nome bonito para site.
HTML (HyperText Markup Language) é a linguagem padrão para a criação de páginas web. Ele usa uma estrutura de tags para definir o conteúdo e a estrutura de uma página.
As tags são elementos que indicam ao navegador como exibir o conteúdo, como h1 para títulos, p para parágrafos, a para links, e img para imagens. Elementos básicos de uma página HTML incluem:

html: A raiz do documento HTML.

head: Contém metadados, links para estilos (CSS), scripts (JavaScript), e o título da página.

body: Contém o conteúdo visível da página, como texto, imagens, e outros elementos.

HTML é a tecnologia que fala o que é cada item em uma página de internet / um site. E ele fez isso por meio de TAG's. TAG's ? Tag's são etiquetas. São os trechos de código que nos ajudam a falar o que é o que em uma página.

HTML é fundamental para a criação de qualquer página web e trabalha em conjunto com CSS e JavaScript para criar páginas web interativas e estilizadas.

Principais Tags HTML:

Estrutura Básica

    html: Define o documento como HTML.

    head: Contém metadados e links para estilos e scripts.

    body: Contém o conteúdo visível da página.

Formatação de Texto:

    h1 a h6: Definem cabeçalhos, sendo h1 o mais importante e h6 o menos.

    p: Define um parágrafo.

    a: Cria um hyperlink. Exemplo: Link

    strong e em: Destacam texto, deixando em negrito e itálico, respectivamente.

Listas:

    ul: Cria uma lista não ordenada.

    ol: Cria uma lista ordenada.

    li: Define um item de lista.

    img: Insere uma imagem. Exemplo:
